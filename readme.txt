viewAttributes

(PLUGIN PARA QGIS)

Plugin creado para la visualizacion en un formulario de informacion acerca de una parcela seleccionada, dando informacion sobre ella y mostrando una imagen almacenada en la carpeta fotos donde se encuentre el archivo shp.
La ruta a la imagen es un campo del shapefile.
